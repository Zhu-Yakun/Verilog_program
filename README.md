31条指令CPU由于没有经验，所以导致文件有点污染，所以索性把前仿真、后仿真、下板分开了  
>> CPU31 -> 前仿真  
>> CPU31_zyk -> 后仿真  
>> CPU31_Board -> 下板  

建议做之前仔细看一下我的31条CPU的报告的最后，相信你会回来感谢我一下（比如点个Star）
